# Online Payments Fraud Detection using Machine Learning

This project focuses on building a Machine Learning system to detect fraudulent online payment transactions using real-world financial data.

## 🚀 Project Overview

Online payment fraud is a major challenge in the digital financial sector.  
This project aims to analyze transaction data and build a predictive model that can classify transactions as **fraudulent or legitimate**.

## 📊 Workflow

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training using Machine Learning algorithms
- Model Evaluation and Comparison

## 🧠 Machine Learning Models Used

- Logistic Regression  
- Decision Tree  

## 📈 Evaluation Metrics

- Accuracy  
- Precision  
- Recall  
- F1-Score  
- ROC-AUC Curve  

## 🔧 Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  

## 🎯 Key Insights

- Identified patterns in fraudulent transactions  
- Improved detection performance using feature engineering  
- Built a reliable classification model for fraud prediction  

## 📌 Goal

To demonstrate practical skills in Data Science and Machine Learning by solving a real-world financial fraud detection problem.
